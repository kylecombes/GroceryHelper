The Big Idea of GroceryHelper

Grocery shopping can often become a long and cumbersome process. Often times one store will not have what you need which requires you to travel to multiple in order to acquire all of the groceries that you need. GroceryHelper is a tool to make grocery shopping quicker and easier. When ingredients and an origin location are inputted, our GroceryHelper web app will output a shopping plan which includes: a list of stores to visit in sequential order, a map with directions to each store and back to your origin, and a list of which ingredients to buy at each store.

Setting Up GrocerHelper

Getting the Source and Installing Dependencies

In order to run GroceryHelper, you'll need Python 3 installed as well as a few dependencies:

  • Flask
  • untangle
  • requests

If you already have Python 3 installed, just run the following command in Terminal: pip3 install flask untangle requests. Once that is installed, clone the Git repo.

Downloading the Store and Item Data

Once you have the source code and the necessary dependencies installed, you'll next need to download all of the stores in your area. To do this, simply run (in Terminal) python3 setup.py.

When that has finished, run python3 webapp_flask.py to launch the app! Then visit http://127.0.0.1:5000/ in your web browser.
