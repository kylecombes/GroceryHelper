The Big Idea of GroceryHelper:

Grocery shopping can often become a long and cumberson process. Often times one store will not have what you need which requires you to travel to multiple inorder to aquire all of the groceries that you need. GroceryHelper is a tool to make grocery shopping quicker and easier. When ingredients and an origin location are inputted, our GroceryHelper webapp will output a shopping plan which includes: a list of stores to visit in sequential order, a map with directions to each store and back to your origin, and a list of which ingredients to buy at each store.

Running GrocerHelper

A hosted version of GroceryHelper can be accessed at https://groceryhelper-sd2017.herokuapp.com/ In order to run GroceryHelper, you'll need to install a few Python 3 dependencies. Just run the following command in Terminal: pip3 install flask untangle requests. Then to launch the web app, run python3 webapp_flask.py.
